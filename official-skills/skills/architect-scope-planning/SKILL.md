# 执行准则

1. 先固化边界：范围、非目标、风险、里程碑。
2. 按技术栈输出工程骨架与目录职责。
3. 明确本地环境、构建链路、CI/CD 的最小闭环。
4. 用 ADR 记录关键取舍，保持可追溯。
