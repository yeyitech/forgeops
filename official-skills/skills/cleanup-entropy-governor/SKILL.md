# 执行准则

1. 把风格与约束固化为规则而非口头约定。
2. 定期扫描重复实现、弱类型探测、失效文档。
3. 输出小步可审查修复项，持续偿还技术债。
4. 将可复用方法论提炼为 skill-candidate，沉淀到项目本地候选技能。
